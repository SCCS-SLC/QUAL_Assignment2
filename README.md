# QUAL_Assignment2

### running: 
first, you have to install the node modules, so run npm install from command prompt or use the *install.bat* file, which will do it for you  

then, to run this program, you simply open the root directory in command prompt, then type **"node main"**  
alteratively, you can run the attached *run.bat* file which will just do it for you
###### afterwards, please type "help" as it will give you the list of commands, but you can just refrence this too



### -- COMMANDS -- 
addEvent -> adds an event  
addAttendee -> adds an attendee  
printAll -> gives the info on a specific event (in JSON format):
- Event name
- count of people attending
- max capacity
- list of people in attendance
  
help -> help command   
exit -> stops execution
